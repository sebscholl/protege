# Protege Guide

Public user-facing documentation for running and operating Protege.

This directory is intentionally separate from `docs/`, which is reserved for internal planning, ADRs, and engineering conventions.

## Guides

1. `guide/cli.md`: CLI command reference and examples.
2. `guide/relay.md`: Relay setup and operational usage.
3. `guide/troubleshooting.md`: Common issues and quick fixes.
